# Knowledge Base

Pagify's growing collection of best practices, evaluations, and patterns for building real websites for real people.

## Structure

| Directory | What's in it |
|---|---|
| `hosting/` | Evaluated hosting platforms, domain registrars, and analytics options — free tiers, tradeoffs, deployment guides |
| `seo/` | SEO patterns, meta tag templates, structured data examples |
| `geo/` | Generative Engine Optimization — how to structure for AI search |
| `architecture/` | Site structure patterns, multi-page conventions, file organization |
| `security/` | HTTP security headers, CSP configuration, and hardening guides |
| `agent-patterns/` | How Claws/agents interact with sites — maintenance, content updates |

## Contributing

Each entry should be:
- **Practical** — usable immediately, not theoretical
- **Honest** — include tradeoffs, not just benefits
- **Dated** — pricing and features change; note when you last verified
- **Sourced** — link to official docs, not blog posts

When an entry becomes stale (>6 months unverified), mark it with `⚠️ UNVERIFIED` at the top.
