# Agent Patterns

Playbooks for AI agents (Claws) that build, maintain, and update Pagify sites.

## Patterns

| Pattern | When to Use |
|---|---|
| [Scaffold New Site](scaffold-new-site.md) | Taking a client from intake to deployed site |
| [Content Update](content-update.md) | Adding or modifying pages on an existing site |

## Who These Are For

1. **Agents** — follow these step-by-step when executing tasks
2. **Coaches** — use these to verify agent work and understand what "done" looks like
3. **Self-sovereign users** — reference these to do it yourself without an agent

## Contributing

New patterns should:
- Solve a real, recurring task (not theoretical)
- Include a verification checklist (how to know it's done correctly)
- List common mistakes (what agents get wrong)
- Be tested against at least one real site before merging
